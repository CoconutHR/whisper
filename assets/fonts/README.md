# Noto Color Emoji

`Noto-COLRv1.woff2` is a self-hosted web font generated from
`fonts/Noto-COLRv1.ttf` in the Google Fonts `noto-emoji` repository at commit
`8998f5dd683424a73e2314a8c1f1e359c19e8742`.

The source font was converted to WOFF2 with FontTools without subsetting or
changing glyphs. It is distributed under the SIL Open Font License 1.1 in
`NotoEmoji-LICENSE.txt`.
